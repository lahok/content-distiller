# Knowledge Purifier 知识提纯智能体

基于 Astron Agent 搭建的通用短文知识提纯智能体，可自动处理短文内容，提取核心信息并输出结构化精简结果，助力高效获取关键知识。

## 项目介绍

本项目是基于 Astron Agent 平台开发的智能体应用，核心目标是解决海量短文信息中关键内容提取效率低的问题。通过智能体的自动化处理能力，快速过滤冗余信息，提炼核心观点、关键要点、案例数据等结构化内容，输出简洁易懂的知识摘要，适用于学习资料整理、信息调研、文档精简等多种场景。

本仓库开源智能体的 YAML 配置文件，可直接在 Astron Agent 平台复现部署，无需重复开发核心逻辑。

## 核心功能

- 自动处理短文：支持输入各类通用短文，无需手动预处理

- 核心信息提取：精准提取文章核心观点、关键要点、重要案例及数据

- 结构化输出：以清晰的格式呈现精简后的知识内容，便于阅读和使用

- 高效复用：配置文件可直接在 Astron Agent 平台部署运行，快速落地使用

## 部署要求

- 已部署并可正常访问的 Astron Agent 平台（参考 Astron Agent 官方部署文档）

- 支持读取 YAML 配置文件的 Astron Agent 版本（建议使用最新稳定版）

## 部署步骤

1. ### 下载配置文件
        从本仓库下载 agent-config.yaml 文件至本地

2. ### 导入 Astron Agent 平台
        1. 登录已部署的 Astron Agent 平台2. 进入「智能体管理」页面，点击「导入智能体」按钮3. 选择本地下载的 agent-config.yaml 文件，完成导入

3. ### 调试与运行
        1. 导入完成后，在智能体列表中找到「知识提纯智能体」2. 点击「调试」按钮，输入测试短文内容，验证智能体是否正常提取核心信息3. 调试通过后，点击「运行」即可正式使用

## 使用说明

1. 在智能体运行页面，输入需要处理的短文内容（建议单篇字数控制在 2000 字以内，提升处理效率）

2. 点击「开始处理」，等待智能体完成信息提取与精简

3. 查看输出结果，可直接复制使用或导出保存

## 注意事项

- 输入短文需为文本格式，避免包含特殊符号或杂乱代码，否则可能影响提取效果

- 若部署后出现功能异常，建议检查 Astron Agent 版本是否兼容，或重新导入配置文件

- 如需扩展功能（如支持长文档、特定领域文本提纯），可基于本配置文件在 Astron Agent 平台进行二次开发
